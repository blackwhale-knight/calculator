# Calculator

# 需求
> 設計一個計算機 (Calculator)，可以執行兩個數的加、減、乘、除計算， 可以進行 undo 和 redo 操作，在實現功能的基礎上進行最佳化設計，備註：undo 和 redo 就是 復原和取消 (上次) 復原的操作。 

# 功能
### 加法
```shell
$ add 1000
```

### 減法
```shell
$ subtract 100
```

### 乘法
```shell
$ multiply 1.5
```
### 除法
- 不能除以 0
```shell
$ divide 20
```

### Undo
```shell
$ undo
```

### Redo
```shell
$ redo
```

### 退出
```shell
$ exit
```
